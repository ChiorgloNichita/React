# Проект Mini-Blog (Версия с использованием CSS-файлов)

## Описание
В этой версии проекта используется внешний CSS-файл для стилизации компонентов. Все стили размещаются в одном файле `App_v1.css`, который подключается в компоненте `App.jsx`. Это один из традиционных способов стилизации в React-приложениях, где стили определяются в одном месте и применяются глобально.

## Структура проекта
Проект состоит из нескольких компонентов:
- **`App.jsx`**: основной компонент, который включает в себя `Header`, `ArticleList` и `Footer`.
- **`App_v1.css`**: файл с внешними стилями для всех компонентов.
- **`index.css`**: глобальные стили.
- **`index.html`**: основной HTML файл, в который загружается React-приложение.
- **`main.jsx`**: точка входа в приложение.

## Код и объяснение

### 1. **App.jsx** (главный компонент)

```jsx
import './App_v1.css'; // Подключаем стили из внешнего CSS-файла
import Header from './components/Header';
import Footer from './components/Footer';
import ArticleList from './components/ArticleList';

export default function App() {
  return (
    <>
      <Header />
      <main>
        <ArticleList />
      </main>
      <Footer />
    </>
  );
}
```

- В этом компоненте происходит подключение внешнего CSS-файла с помощью импорта:
  ```jsx
  import './App_v1.css'; 
  ```
  Это позволяет применить стили, определенные в `App_v1.css`, ко всем компонентам в приложении.
  
- Компоненты `Header`, `ArticleList`, и `Footer` встраиваются в основной контейнер `App`, который будет отображать основную структуру страницы.

### 2. **App_v1.css** (файл стилей)

```css
#root {
  max-width: 1400px;
  margin: 0 auto;
  padding: 2rem;
  text-align: center;
}

.header {
  display: flex;
  color: rgb(10, 50, 160);
  padding: 0.7rem;
  justify-content: center;
  gap: 1rem;
  border-radius: 12px;
  border: 1px solid #88a;
  font-family: 'Space Grotesk', sans-serif;
}

.header__title {
  margin: 0;
  font-size: 2rem;
}

.article {
  font-family: 'Times New Roman', sans-serif;
}

.article__title {
  color: rgb(20, 40, 120);
}

.footer {
  font-size: small;
  font-family: 'Space Grotesk', sans-serif;
  color: rgb(10, 50, 160);
}
```

- В `App_v1.css` определены стили для различных компонентов:
  - `#root`: задает максимальную ширину и центрирует содержимое страницы.
  - `.header`: стилизует компонент заголовка, применяя цвета, отступы и оформление.
  - `.article`: задает стиль для статей, включая шрифт и цвет заголовков.
  - `.footer`: стилизует футер, задавая размер шрифта и шрифт для текста.

Каждому классу в CSS-файле соответствует компонент в React (например, `.header` для компонента `Header`, `.article` для компонента `Article` и т. д.).

### 3. **index.css** (глобальные стили)

```css
:root {
  font-family: Inter, system-ui, Avenir, Helvetica, Arial, sans-serif;
  font-weight: 400;
}
```

- Глобальные стили задают начальные параметры для шрифтов и шрифта по умолчанию на всей странице. Эти стили будут применяться ко всем компонентам, если для них не указаны другие шрифты.

export default function Footer() {
